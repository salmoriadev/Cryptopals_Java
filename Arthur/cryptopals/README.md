## 📖 Sobre o Projeto

Este repositório contém minhas soluções para os [Desafios de Criptografia Cryptopals](https://cryptopals.com/), implementadas em **Java** com o auxílio da biblioteca criptográfica **Bouncy Castle**.

O objetivo principal deste projeto é o aprendizado e aprofundamento em conceitos de criptografia na prática, explorando desde cifras clássicas até ataques mais complexos a primitivas criptográficas modernas.

## 🚀 Motivação

- **Aprender Criptografia Aplicada:** Ir além da teoria e implementar algoritmos e ataques criptográficos.
- **Aprimorar Habilidades em Java:** Utilizar recursos modernos da linguagem e boas práticas de desenvolvimento.
- **Explorar o Bouncy Castle:** Dominar uma das bibliotecas criptográficas mais poderosas e amplamente utilizadas no ecossistema Java.

## 🛠️ Tecnologias Utilizadas

- **[Java (JDK 17+)](https://www.oracle.com/java/technologies/downloads/)**: Linguagem de programação principal.
- **[Maven](https://maven.apache.org/)**: Ferramenta para gerenciamento de dependências e build do projeto.
- **[Bouncy Castle](https://www.bouncycastle.org/java.html)**: Biblioteca para primitivas e algoritmos criptográficos.
